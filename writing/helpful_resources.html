Helpful resources

We recommend the following external guides for further documentation training:

  • Google Technical Writing Course: This two-part course teaches the basics of technical writing for engineering audiences. To run this course, designate one person as the instructor and have them use the Facilitator's guide.
  • I'd Rather Be Writing: This comprehensive guide covers the ins and outs of APIs, from documentation to publishing and metrics.
  • Swagger: Learn how to write comprehensive API documentation that conforms to the OpenAPI 3.0 specification.
